■使用方法

-1. ダウンロードしたファイルを解凍
-2. setting.ini.phpをメモ帳で開いて設定を書き換え
-3. 転送してパーミッションを変更
-4. 転送先へブラウザでアクセス

■パーミッションの変更について

- db ディレクトリのパーミッションを 777 に変更
- *.php ファイルのパーミッションを 777 に変更
- その他 644 になっていれば動く
